# Sounds

Put your local soundpad audio files here when testing OBS ControlDeck.

Supported examples:

```txt
sounds/bruh.mp3
sounds/vine-boom.mp3
sounds/laugh.wav
sounds/alert.wav
sounds/transition.wav
```

Do not commit copyrighted sound files unless you have permission to publish them.

Recommended open-source workflow:

- keep only placeholder docs in the repository;
- add your personal audio files locally;
- share sound pack JSON files separately;
- include license info for any public audio assets.
