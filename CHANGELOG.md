# Changelog

All notable changes to OBS ControlDeck will be documented in this file.

## 1.0.0 - Stable Lua release

### Added

- Stable OBS Lua script release.
- Eight configurable Soundpad slots.
- AutoMarker for timestamped recording markers.
- ClipNotes-style marker categories.
- Reaction marker hotkey.
- Scene Timer session tracking.
- Panic Button workflow.
- Recording Guard checks.
- AutoIntro scene flow.
- JSON and TXT marker export.
- Config export.
- Safe config import preview.
- Preset schema documentation.
- Release notes and release checklist.
- Security policy.
- GitHub issue templates.

### Changed

- README now describes the project as a stable v1.0.0 foundation.
- Script version updated from `0.1.0` to `1.0.0`.
- Soundpad expanded from four slots to eight slots.
- AutoIntro timer handling improved.
- Session saving now avoids duplicate save attempts.

### Removed

- YouTube helper concept is intentionally excluded.
- No bundled copyrighted sound files.
- No silent third-party config overwrite behavior.

## 0.1.0 - Initial scaffold

### Added

- Project concept and README.
- OBS Lua script foundation.
- Marker and clip note export design.
- Scene timer design.
- Panic Button design.
- Recording Guard design.
- Basic Soundpad design.
- Config import/export design.
- Example presets and documentation.
